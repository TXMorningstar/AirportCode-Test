运行“考试程序.exe”即可快速开始考试进行自测；
本程序的题库可以在“database.txt”中进行增、删、改、查，注意一共有两个“database.txt”，每个“database.txt”都只对同一级文件夹内的考试程序生效；
本程序的源码为airport_code_map.py，更改源码并不会影响“考试程序.exe”的运行，如果需要运行自行修改的代码，请使用python解释器或自行打包。

建议使用姿势：编辑database中的内容，留下十到二十道你想要记忆的三字代码题，然后（像考科目一前拼命刷驾考宝典的学徒一样）反复刷题记忆。
